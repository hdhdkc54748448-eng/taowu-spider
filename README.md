梼杌爬虫架构已经开发完成，稳定版v6.0已发布后续会推出英语版本

爬虫软件仅用于教学与实践，切莫用于非法途径，本人开发出来只是为了教学实践，如若用于恶意用途，后果自负

设计之初ui渲染较卡顿，我拉大了框体增强流畅度运行时后台不要运行过多软件尤其是在浏览器增强模式时

使用方法如下：
1、解压文件

2、双击梼杌v1.0.exe

3、弹出界面后，在存储模式旁边的下拉框选择用内置数据库还是本地MySQL，如果使用mysql要对照好库名和密码，如图所示，如果是内置数据库不用填密码。
<img width="1191" height="756" alt="屏幕截图 2026-06-13 151251" src="https://github.com/user-attachments/assets/ef841f46-5937-4956-b1c1-93ad90553748" />


4、选择完数据库模式后，粘贴招聘网站详情页至第一个大框，粘贴完后点击测试链接，之后点击下方的按钮，一共两个模式通用抓取和浏览器模拟抓取，自行选择，浏览器增强模式启动后会自动弹出页面这时千万一定不要更不该碰它，当看到页面缓缓下拉至底部后会自动关闭，如果抓取失败则不会，通用模式则是静默的没有页面的也不要碰它，运行时紧盯终端。

5、按下模式后抓取完后数据会在终端窗口弹出日志会看到文件路径
<img width="1485" height="672" alt="屏幕截图 2026-06-13 151801" src="https://github.com/user-attachments/assets/996cc2eb-b12a-45c7-ac5c-7516f7c77c5a" />

6、选择那个.json文件格式的那一条复制，粘贴在抓取后弹出的页面顶端的文件路肩路径一栏点击旁边的按钮，如图中所示的翠绿色按钮解析入框后
<img width="1870" height="1058" alt="屏幕截图 2026-06-13 150548" src="https://github.com/user-attachments/assets/20ef20a8-f819-4f47-87fd-3dafef37712f" />


7、来到下一行，选择条目是指定当前条目进行操作，可一次解析一个文本入框，然后右边有五个按钮，模型抽取是自动抽取精度不足，旁边有手动关键词抽取进行补充，当然你想偷懒也可以点击黄色按钮直接入库，后面两个按钮分别是是指定条目和清空文本框的。

8、模型抽取的结果会直接输入左下方的录入区，从左至右功能是直接入库，新增条目，清空输入

9、右下角的数据表区是实时查看数据表情况的可以刷新

10、本软件不能用于而已用途，本人再次声明，如果用于恶意用途，后果自负，本人开发后用于教学实践。
